03 / 09 GAMEPLAY 他會做什麼？

寫一行 Python，
Agent 就在世界裡動起來。

孩子實際在課堂上做的事 — 用 Python 對麥塊 Agent 下指令， 建立、修復、自動化整個 Minecraft 世界。

▸ AGENT · 自動挖礦

讓 Agent 自己挖一條 100 格的礦道

用迴圈和條件判斷，讓 Agent 邊挖邊撿、遇到熔岩自動轉向。一段 Python 程式碼搞定。

# 讓 Agent 挖 100 格
for i in range(100):
    agent.destroy("forward")
    agent.move("forward")
▸ BUILD · 自動建造

用迴圈蓋出一座 10×10 城堡

不再一塊一塊放方塊。用 Python 寫個函式，幾秒內就讓 Agent 把整座城堡蓋完。

# 一秒蓋一面牆
def build_wall(length):
    for _ in range(length):
        agent.place_block()
▸ REPAIR · 修復世界

偵測被炸毀的區塊、自動修復

Creeper 把房子炸掉了？用 Python 寫一個「修復腳本」，Agent 自動補回所有被炸掉的方塊。

# 偵測 + 修補
if block_missing():
    agent.replace("stone")
▸ PET · 自製寵物

寫一隻會跟著你走的小狗

用 Python 控制座標、追蹤距離。寫完，麥塊裡就多了一隻只屬於孩子自己的寵物。

# 跟隨主人
while True:
    pet.follow(player)

09 / 09 INVENTORY 家長最常問的 12 個問題

裝備你的
常見問題包。

  1. FILE.01 麥塊 Python 跟一般的 Python 課差在哪？ +
    差在「動機來源」。傳統 Python 課的孩子看不到立即效果，容易放棄。麥塊 Python 寫一行程式 Agent 就動 — 對 5-9 年級孩子來說，這個即時回饋是「繼續學下去的關鍵」。內容上 80% 是針對 Agent 操作的 Python（產生在麥塊世界），20% 是通用 Python 基礎。
  2. FILE.02 我孩子沒上過橘蘋的麥塊基礎班可以來嗎？ +
    可以，只要 5 年級以上、有玩過麥塊。我們會在體驗課時讓老師現場評估「打字速度」和「麥塊操作熟悉度」，確認能跟上節奏即可加入。沒上過基礎班不會被擋。
  3. FILE.03 為什麼是 5-9 年級這麼大的範圍？ +
    因為麥塊 Python 的設計是「降低門檻」的延續課，不是分齡的能力班。5 年級剛上完麥塊基礎想繼續、9 年級拒絕傳統 Python 想換切入點 — 都適合。
  4. FILE.04 線上課我孩子能跟上嗎？ +
    橘蘋線上麥塊體系已運作 100+ 班、累積近 3,000 位學生，線上課程設計成熟。每堂課採雙師制：影片中由資深講師專業講解課程內容，同時線上有一位真人老師 1 對 1 即時回答孩子的問題。發現孩子卡住會立刻接住，不會因為線上就被忽略。
  5. FILE.05 缺課怎麼辦？調課有限制嗎？ +
    線上課程沒有「補課」這個概念，而是彈性調課。家長直接到橘蘋官網的家長平台自助操作，每一堂課都能調整時段（最晚前一天 24:00 前完成，可調入未來 3-60 天內任何時段），沒有次數上限，不必通知專員。唯一的條件：需在該期上課期限內完成所有課程（例：購買 1 期 15 堂，須在 15 週內上完）。
  6. FILE.06 一定要連上 3 期 45 堂嗎？ +
    不一定。每期 15 堂為一個獨立階段，可以單期報名、上完再決定要不要續報。完整的 3 期下來，孩子能走完一條完整的學習弧線 — 從 Python 基礎 → 自動化創作 → 完整個人作品。每一期都有屬於那個階段獨特的成就感與作品，3 期合起來，孩子帶走的是一份完整的麥塊程式創作經驗。
  7. FILE.07 上完真的能寫 Python 嗎？還是只能在麥塊裡？ +
    80% 麥塊 Agent 操作 + 20% 通用 Python 基礎。完課後孩子有「真實寫過 Python」的經驗、認識變數迴圈條件等核心概念。但如果要往更深的 Python 程式（資料結構、演算法）走，需要再上 Python 程式開發班補齊。不能跳過 Python 程式開發班。
  8. FILE.08 完課後一定要接 Python 程式開發班嗎？ +
    完全不用。麥塊 Python 的設計初衷是「讓孩子在熟悉的麥塊世界裡，延續對寫程式的興趣與探索」。完課的孩子有真實程式經驗、有作品集、有信心 — 想繼續就有選擇，不想繼續也不是失敗。如果想往程式深處走，可以接 Python 程式開發班（需通過評估測驗）；如果對其他科技領域有興趣，橘蘋還有 AI 思維、ITS 國際認證 等課程可以延續學習。每條路都通往未來。
  9. FILE.09 老師是誰？會固定嗎？ +
    橘蘋有 100+ 班的雙師制師資，所有老師都通過橘蘋教學部嚴格的篩選與培訓。固定班級會有固定的影片講師與線上真人老師搭配。我們不主打「明星老師」，主打的是「整套橘蘋教學品質保證」 — 不管孩子被分到哪位老師，都是同一套標準。每一位老師都有豐富的兒少程式教學經驗、最懂得用孩子聽得懂的方式引導，有耐心陪孩子走過每一個卡關時刻。
  10. FILE.10 家長需要懂程式或麥塊嗎？要陪上嗎？ +
    完全不用。孩子上課時家長可以做自己的事，雙師制（影片講師 + 線上真人老師）會照顧孩子的上課狀態。每週會有學習週報，家長隨時可以查看孩子的學習進度與狀況；每期結束還會有完整的作品展示與進度回饋，家長一看就知道孩子做了什麼、學了什麼。
  11. FILE.11 需要什麼設備？電腦規格要很高嗎？ +
    一台能跑 Minecraft 教育版的電腦 + 網路就夠了。手機 / 平板無法替代（程式碼編輯需要鍵盤）。具體規格諮詢時會給家長完整清單。報名前我們會協助確認設備能不能跑。
  12. FILE.12 價格 / 招生時程怎麼了解？ +
    留下諮詢資料，專員會主動聯繫。會依您孩子的年級、麥塊背景，給出最適合的開班時段與費用方案。建議先預約一堂免費體驗課，讓孩子實際感受麥塊 Python 的學習氛圍。
